    업무 균등 분산 (섀넌 엔트로피)
    ?
    [업무 균등 분산 (섀넌 엔트로피)]
    • 업무의 다양성과 분산을 판단
    • 수식 : H = -∑ pi log2(pi)
    • 0.0~0.5 : 단순 명확한 업무(Deep Work), 80%이상의 시간을 특정업무에 전담
    • 0.5~1.5 : 주업무 중 간간히 회의 등을 병행하는 일반적인 상태
    • 1.5~2.5 : 업무 파편화 가능성, 맥락 전환 비용 상승
    • 2.5이상 : 정신없음. 번아웃 위험
    업무 쏠림도 (HHI 지수)
    ?
    [업무 쏠림도 (HHI 지수)]
    • 특정 업무, 인원에게 얼마나 쏠려 있는가(독점/집중)를 수치로 환산
    • 수식 : HHI = ∑ (si)2
    • 0.15 미만 : 저집중. 안정적이나 핵심 부족 가능성
    • 0.15~0.25 : 핵심업무에 적절히 리소스 배분
    • 0.25~0.40 : 특정 업무/인원 의존도 높음
    • 0.40 초과 : 초고집중. 특정 업무나 인원에 극도로 쏠림. (병목)
    전체 업무 변동성 (CV 추이)
    ?
    [전체 업무 변동성 (CV 추이)]
    • 업무의 양과 내용이 주차별/일별로 얼마나 일관성 있게 유지되는지 판단
    • 수식 : CV = σ / μ
    • Low(0.3 미만) : 업무량이 일정. 루틴이 정해져 있음
    • High(1.0 이상) : 어떤 날은 8시간 내내 한 일만 하다가, 어떤 날은 수십 가지 일을 번갈아 함
