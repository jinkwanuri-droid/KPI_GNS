    업무 과부하 지수 (OT Index)
    ?
    • 정의 : 일일 근무 시간의 가중치 합산 평균.
    • 로직 : 8-9h(0.5), 9-10h(0.75), 10h+(1.0) 가중치 부여 후 업무일수로 나눔.
    • 목적 : 단순 야근 횟수가 아닌 노동의 강도를 측정하여 번아웃 리스크 탐지.
    업무 파편화도 (Shannon Entropy)
    ?
    • 정의 : 정보 이론의 엔트로피 개념을 활용한 업무 복잡도 측정.
    • 로직 : -∑p_i*ln(p_i)를 최대 엔트로피로 정규화. 1에 가까울수록 업무가 잘게 쪼개짐.
    • 목적 : 맥락 전환(Context Switching) 비용을 파악하여 몰입 방해 요소를 진단.
    핵심 업무 집중도 (HHI)
    ?
    • 정의 : 허핀달-히르슈만 지수를 응용한 업무 점유율 분석.
    • 로직 : 각 업무 비중의 제곱합. 1에 가까울수록 특정 과업에 몰입함.
    • 목적 : 잡무를 제외하고 실제 '선택과 집중'이 일어난 핵심 성과 영역을 추출.
    업무 안정성 (CV)
    ?
    • 정의 : 투입 시간의 변동계수(CV=σ/μ)를 활용한 성실도 지표.
    • 로직 : 1 - (CV / Max_CV)로 정규화하여 1에 가까울수록 기복이 적음을 의미.
    • 목적 : 매일 정해진 페이스를 유지하는 '에너지 공급의 균일성'을 평가.
    업무 리듬 주도성 (Hurst Exponent)
    ?
    • 정의 : 시계열 데이터의 장기 의존성을 측정하는 지수.
    • 로직 : 0.5를 기준으로 1에 가까울수록 강한 추세(안정적 리듬), 0에 가까울수록 무작위성(혼돈).
    • 목적 : 돌발 상황에 휘둘리지 않고 스스로 업무 리듬을 통제하는 자기 주도성 측정.
    직무 확장성 (Jaccard Similarity)
    ?
    • 정의 : 집합 간 유사도 측정을 통한 업무 도메인의 변화율 분석.
    • 로직 : (과거 업무 집합 ∩ 현재) / (과거 ∪ 현재).
    • 목적 : 1에 가까우면 숙련도 향상(전문가형), 0에 가까우면 새로운 도전(성장형).
