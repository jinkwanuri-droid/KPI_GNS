The Challenge

도면의 오차는 mm 단위로 잡으면서,
사람의 성과는 눈대중으로 재고 있지 않습니까?

분기말의 희미한 기억에 의존하는 평가, 매일 버려지는 소중한 업무 기록들. 감(感)에 의존하는 평가는 이제 끝내야 합니다.

Core Philosophy

성장을 위한 3단계 프로세스

01
조용한 데이터 추출
기존 업무일지를 그대로 사용합니다. 새로운 입력이나 방해 없이 워크플로우에 녹아듭니다.
02
감(感)을 수치(Data)로
직관적인 업무 투입시간, 야근 빈도 등을 수치화하여 피드백의 탄탄한 기초를 마련합니다.
03
숨겨진 인사이트 도출
수학적 모델을 통해 업무 패턴, 리소스 밸런스를 입체적으로 분석하고 시각화합니다.

6 Core Metrics

성과를 측정하는 수학적 기준

Index 01
섀넌 지수
파편화도 · 질적 몰입도
업무가 얼마나 여러 갈래로 분산되었는지 파악하여, 몰입을 방해하는 요소를 찾습니다.
H = −Σ pᵢ ln pᵢ
Index 02
허스트 지수
주도성 · 질적 통제력
수동적 업무 처리가 아닌, 팀원 스스로 업무 리듬을 끌고 나가는 주도성을 측정합니다.
E[R/S] = C nᴴ
Index 03
자카드 지수
확장성 · 질적 성장성
반복 업무의 늪에 빠지지 않고, 새로운 영역을 탐색하고 있는지를 나타내는 성장의 궤적입니다.
J = |A ∩ B| / |A ∪ B|
Index 04
초과근무 지수
정시업무 · 양적 에너지 잔량
총 근무 중 초과근무 비율을 역산하여 건강한 에너지 상태와 번아웃 위험을 진단합니다.
Idx = 1 − (T_ot / T_total)
Index 05
허핀달-허쉬만 지수
몰입도 · 양적 화력 집중도
특정 핵심 프로젝트에 에너지를 온전히 집중하고 있는지를 수치화합니다.
HHI = Σ sᵢ²
Index 06
변동계수
안정성 · 양적 투입 리듬
일일 업무 투입 시간의 기복을 측정하여 심리적 불안도와 안정적인 업무 리듬을 파악합니다.
CV = σ / μ
