Core Philosophy

3단계 성장 프로세스

새로운 입력 없이, 기존 데이터에서 인사이트를 자동 추출합니다.

01

조용한 데이터 추출

기존에 쓰던 업무일지 그대로 GNS가 조용히 데이터를 추출합니다. 현장의 흐름을 방해하지 않고 워크플로우에 자연스럽게 녹아듭니다.

02

감(感)을 수치(Data)로

Shannon, Hurst, Jaccard 등 검증된 수학적 모델을 통해 업무 패턴과 리소스 밸런스를 입체적으로 분석하고 수치화합니다.

03

팩트 기반의 성장

막연한 말 대신 명확한 데이터로 다음 분기를 설계합니다. 팩트에 근거한 성장 계획은 구성원 모두에게 투명한 신뢰를 제공합니다.

01

데이터의 선순환 구조

업무일지를 제대로 쓴다 ➔ 데이터 품질 향상 ➔ 정확한 피드백 ➔ 납득 가능한 평가 ➔ 작성 동기 부여로 이어지는 선순환을 만듭니다.

02

업무 패턴 분석

개인별 업무 집중 영역 및 변화 추이 식별하고, 팀 포트폴리오 시각화를 통해 팀 전체 리소스 배분의 균형점을 확인합니다.

03

팩트 기반의 성장

막연한 말 대신 명확한 데이터로 다음 분기를 설계합니다. 팩트에 근거한 성장 계획은 구성원 모두에게 투명한 신뢰를 제공합니다.

6 Core Metrics

핵심 KPI 지표

수학적으로 검증된 6가지 지표로 성과의 전 방위를 측정합니다.

Shannon Entropy Index
섀넌 지수
파편화도 · 질적 몰입도
업무의 연속성과 질적 집중도를 측정합니다. 정보 엔트로피 이론을 기반으로 업무가 얼마나 다양하게 분산되었는지를 수치화합니다.
H = −Σ pᵢ ln pᵢ
데이터: 각 프로젝트(업무)별 투입 시간의 비율(pᵢ)
계산: 해당 비율과 그 자연로그 값을 곱한 뒤 모두 더하여 음수로 변환합니다.
의미: 수치가 클수록 한 가지 일에 집중하지 못하고 여러 업무로 파편화되어 있음을 뜻합니다.
Hurst Exponent
허스트 지수
주도성 · 질적 통제력
업무가 외부 요청에 의해 휘둘리는지, 아니면 팀원이 스스로의 리듬대로 끌고 가는지(지속성)를 보여줍니다. 높을수록 '업무 주도권'을 쥐고 있음을 뜻합니다.
E[R/S] = C nᴴ
데이터: 일정 기간(n) 동안의 업무 투입량 시계열 데이터
계산: 누적 편차의 범위(R)를 표준편차(S)로 나눈 값이 시간의 흐름에 따라 증가하는지 회귀 분석합니다.
의미: H > 0.5이면 팀원이 스스로 리듬을 유지하며 주도적으로 가속도를 붙이고 있음을 뜻합니다.
Jaccard Index
자카드 지수
확장성 · 질적 성장성
단순히 같은 일을 반복하는지, 아니면 새로운 기술이나 영역을 탐색하고 있는지를 보여주는 '성장의 궤적'입니다. 지표 정체는 매너리즘을 시사합니다.
J = |A ∩ B| / |A ∪ B|
데이터: 어제의 업무 키워드 집합(A)과 오늘의 업무 키워드 집합(B)
계산: (두 날 모두 수행한 업무 수) / (두 날 중 한 번이라도 수행한 전체 업무 수)
의미: 0에 가까울수록 어제와 다른 새로운 일을 했다는 뜻이며, "새로운 영역으로의 확장성"을 측정합니다.
Overtime Index
정시업무 지수
정시업무 · 양적 에너지 잔량
초과근무 비율을 역산하여 에너지 잔량을 측정합니다. 지속 가능한 업무 투입의 핵심 지표로, 급격히 떨어지면 '번아웃'에 대한 긴급 케어가 필요함을 뜻합니다.
Idx = 1 − (T_ot / T_total)
데이터: 초과근무 시간(T_ot)과 총 근무 시간(T_total)
계산: 총 근무 시간 중 야근이 차지하는 비중을 구한 뒤 1에서 뺍니다.
의미: 1(100%)에 수렴할수록 번아웃 위험이 적은 건강한 에너지 상태임을 의미합니다.
HHI
허쉬만-허핀달 지수
몰입도 · 양적 화력 집중도
한두 개의 핵심 프로젝트에 에너지를 쏟고 있는지 봅니다. 팀원의 '에너지 효율성'을 보여주며, 가장 중요한 일에 화력을 집중하고 있는지 점검합니다.
HHI = Σ sᵢ²
데이터: 개별 프로젝트에 투입된 시간 점유율(sᵢ)
계산: 각 프로젝트 비중의 제곱을 모두 더합니다.
의미: 특정 업무에 집중할수록 수치가 커집니다. "에너지를 분산하지 않고 한곳에 제대로 쏟았는가"를 보여줍니다.
Coefficient of Variation
변동계수
안정성 · 양적 투입 리듬
업무 투입량이 매일 일정한지 봅니다. 변동폭이 크다면 예측 가능성이 떨어져 심리적 불안도가 높을 수 있습니다. 안정감을 제공하는 근거로 활용됩니다.
CV = σ / μ
데이터: 일정 기간 동안의 일일 업무 투입 시간
계산: 표준편차(σ)를 평균(μ)으로 나눕니다.
의미: 값이 낮을수록 기복 없이 꾸준하게 일했다는 뜻이며, "업무 리듬이 얼마나 규칙적이고 안정적인가"를 진단합니다.

Expected Impact

타겟별 기대효과

모든 구성원이 데이터 기반의 투명한 성장을 경험합니다.

FOR STAFF

투명한 데이터 포트폴리오

내 업무 이력이 투명한 데이터 포트폴리오로 완성됩니다. 평가 시즌에 기억을 더듬을 필요 없이, 시스템이 모든 것을 기록합니다.

FOR LEADER

객관적 데이터 근거 피드백

평가 시즌의 스트레스 해소. 막연한 인상이 아닌 확실한 데이터 근거의 객관적 피드백으로 팀원들과 깊은 신뢰를 구축합니다.

FOR EXECUTIVE

데이터 기반 인력 배치

프로젝트 국면별 투입 리소스를 미리 예측합니다. 데이터를 기반으로 한 전략적 인력 배치로 조직의 성장 역량을 극대화합니다.
